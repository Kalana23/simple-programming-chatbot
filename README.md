## 📌 About The Project
**Learnio Chatbot** is a simple, rule-based conversational agent designed to answer basic and advanced programming-related questions. Built entirely with HTML, CSS, and Vanilla JavaScript, it features a clean, responsive user interface and utilizes keyword-matching algorithms to provide educational responses. 

This project was developed as a part of the **IT1207 | Advanced Data Structures and Algorithms** module.

## ✨ Features
* **Keyword Matching Algorithm:** Scans user input for specific programming-related keywords and returns predefined responses.
* **Modern UI:** A clean, chat-bubble interface styled with CSS (Flexbox, gradients, and responsive design).
* **Instant Replies:** Simulated 400ms delay to mimic a natural conversational flow.
* **Fallback Mechanism:** Handles unknown queries gracefully by prompting the user to ask another programming-related question.
* **Keyboard Support:** Users can send messages simply by pressing the `Enter` key.

## 🛠️ Built With
* **HTML5** - Structure of the chat interface.
* **CSS3** - Styling, animations, and layout.
* **Vanilla JavaScript** - Core logic for keyword matching and DOM manipulation.
